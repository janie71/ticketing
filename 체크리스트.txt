관리자 페이지
• 예매창 여는 시간정할수있도록 설정
*  일단 아에 닫았다가 시간됐을때 열도록
• 시간표 조회 -모든 곡에 대해 시간 삭제, 추가 가능
* 일주일 단위로 보여짐
* 예매화면 선택한 날짜만 공개되도록 함
* 선택한 특정시간은 예약 불가하도록
* 안되는 시간을 선택할수있도록
* 팀 별로 선택한 시간을 볼수있는 필터
• 팀별로 조회
* 삭제, 추가, 색 변경
* 시간 조회

일반유저 페이지
1. 처음화면
• '곡이름선택' 클릭하면 지금까지 만들어진 리스트 중에 선택할수있도록 선택한 곡으로 예매 진행
• 곡 새로 만들기 버튼
* 색깔(한 20개제시)과 이름 작성
* 만들기 버튼 누르면 데이터 추가
• 예매하기 버튼
* 관리자가 설정한 시간되면 활성화되도록
• 곡 합주 일정 조회 버튼

2. 예매화면
• 일주일 단위로 보여지기
• 몇월 몇주차 선택하는 버튼 상단에
• 일~월 30분단위로
• 원하는 시간 클릭하여 예매하기 버튼 누르면 30분단위로 예매가능
• 이미 다른팀이 선택했다면 안된다는 알림
• 예약된 시간은 각 곡별 색깔로 표시
• 자기곡 클릭하면 삭제할거냐는 알림 띄우기 확인누르면 삭제
• 자기 곡에 대해서만 삭제가능

3. 합주 일정 조회
왼쪽 시간표에 색깔로 표시
오른쪽 리스트형태 숫자로 표시
곡이름이 데이터에 없으면 그냥 텅
색깔 변경 가능하도록하기
